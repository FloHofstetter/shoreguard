    Configuration & Exceptions¶

    config ¶

    Shared configuration helpers for Shoreguard.

    is_private_ip ¶

    is_private_ip(host: str) -> bool

    Return True if host resolves to a private/loopback/link-local address.

    Used both at registration time (API validation) and at connection time (DNS-rebinding protection).

    Parameters:

    Name Type Description Default
    host str

    IP address literal or hostname to check.

    		 required

    Returns:

    Name Type Description
    bool bool

    True if the address is private, loopback, link-local, or reserved.

    exceptions ¶

    Domain exceptions for Shoreguard.

    ShoreGuardError ¶

    Bases: Exception

    Base for all Shoreguard domain errors.

    GatewayNotConnectedError ¶

    Bases: ShoreGuardError

    The OpenShell gateway is not reachable or not configured.

    NotFoundError ¶

    Bases: ShoreGuardError

    A requested resource (preset, sandbox, etc.) was not found.

    PolicyError ¶

    Bases: ShoreGuardError

    A policy read or modification failed.

    SandboxError ¶

    Bases: ShoreGuardError

    A sandbox entered an unexpected state.

    ValidationError ¶

    Bases: ShoreGuardError

    Input validation failed (e.g. malformed command syntax).

    FeatureNotAvailableError ¶

    Bases: ShoreGuardError

    A feature is not available in the current OpenShell gateway version.

    Name Type Description Default

    The exception to translate.

    Returns:

    Name Type Description

    list_presets() -> list[dict[str, str]]

    List available policy presets bundled with Shoreguard.

    Returns:

    Type Description
    list[dict[str, str]]

    list[dict[str, str]]: Dicts with keys name, description, and file.

    get_preset ¶

    get_preset(preset_name: str) -> dict[str, Any] | None

    Load a single preset by name.

    Parameters:

    Name Type Description Default
    preset_name str

    Filesystem-safe preset identifier (no path separators).

    		 required

    Returns:

    Type Description
    dict[str, Any] | None

    dict[str, Any] | None: Dict with keys name, description, and policy where policy contains the network_policies / filesystem / process definitions, or None if not found.
